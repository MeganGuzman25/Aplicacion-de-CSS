Introduccion a HTML 5



INICIO ELEMENTOS ATRIBUTOS FORMULARIO WEB



ELEMENTOS HTML

No. Elementos Descripcion
1 a Define un anchor (anclaje o hipervínculo)
2 abbr Marca las abreviaturas del texto y proporciona el significado de esas abreviaturas
3 acronym Marca las siglas o acrónimos del texto y proporciona el significado de esas siglas
4 b Indica que el texto debe ser representado e bold (o negrita)
5 basefont Permite cambiar algunas propiedades del texto
6 bdo Anulación del algoritmo bidireccional (en referencia a la dirección de la escritura)
7 big Muestra el texto marcado con un tamaño de fuente más grande
8 br line break - ruptura (o salto) de línea
9 cite Se emplea para marcar una cita o una referencia a otras fuentes
10 code Delimita el texto considerado un fragmento de código fuente
11 dfn Marca las definiciones de ciertos términos y proporciona el significado de éstos
12 em emphasis – énfasis
13 font Indica el tamaño, color, o fuente del texto que contiene
14 i Muestra el texto marcado con un estilo en cursiva o itálica
15 img Imagen
16 input Posibilita y define la introducción de datos en el formulario
17 kbd Indicar al usuario el texto que debe introducir
18 label Asocia un rótulo o etiqueta a un campo de un formulario
19 q short quotations - cita corta
20 s strike-through - tachado
21 samp Sirve identificar una muestra de los caracteres que forman la salida o el resultado de algún proceso
22 select Crea un contenedor mediante el cual el usuario puede seleccionar de una lista de opciones
23 small Aplica al texto marcado un tamaño de fuente más pequeño
24 span Es un contenedor genérico en línea. Sirve para para aplicar estilo al texto o agrupar elementos en línea
25 strike Muestra el texto tachado con una linea horizontal
26 strong Marca con especial énfasis las partes más importantes de un texto
27 sub Crea un subíndice posicionando el texto marcado por debajo de la linea
28 sup Crea un superíndice posicionando el texto marcado por encima de la linea
29 textarea Crea un control de entrada de texto multilínea
30 tt Representa como texto de teletipo o ancho fijo
31 u Muestra el texto subrayado
32 var Marca variables de programas y similares
33 address Contiene la información de contacto con los autores del documento
34 blockquote Indica que el texto que encierra es una cita textual de otro texto externos
35 center Crea una caja en bloque con el contenido centrado
36 dir Crea listas multicolumna de directorios
37 div Es un elemento en bloque genérico y sirve para crear secciones o agrupar contenidos
38 dl Crea una lista de definiciones
39 fieldset Permite organizar en grupos los campos de un formulario
40 form Actúa como contenedor de controles. Representa un formulario
41 h1, h2, h3, h4, h5, h6 Crea un encabezado o título de primer, segundo, tercer, cuarto, quinto o sexto nivel para una sección del documento respectivamente
42 hr Crea una linea de separación horizontal
43 isindex Crea un control de entrada de texto de una línea
44 menu Crea un menú
45 noframes (sin marcos) - aporta contenidos alternativos a los marcos
46 nos-cript Contenedor de contenido alternativo para la representación no basada en scripts
47 ol Crea una lista ordenada
48 p Párrafo
49 pre Permite que el texto conserve el formato y sea mostrado tal cual
50 table Tabla
51 ul Crea una lista no ordenada
52 dd Descripción de una definición
53 dt Término definido
54 frame-set Subdivisión en ventanas
55 li Objeto de lista
56 tbody Cuerpo de tabla
57 td Celda de datos de una tabla
58 tfoot Pie de tabla
59 th Celda de encabezado de tabla
60 thead Cabecera de tabla
61 tr Fila de una tabla
62 button Botón
63 del Texto borrado
64 iframe Subventana en línea
65 ins Texto insertado
66 map Mapa de imágenes en el lado del cliente
67 object Objeto genérico incluído
68 script Sentencias de script
