INICIO ELEMENTOS ATRIBUTOS FORMULARIO WEB



ATRIBUTOS HTML

No. Atributos Descripcion
1 id="texto" Establece un indicador único a cada elemento.
2 class="texto" Establece la clase CSS que se aplica a los estilos del elemento
3 style="texto" Aplica de forma directa los estilos CSS de un elemento
4 title="texto" Establece el título del elemento (Mejora la accesibilidad)
5 lang="codigo" Indica el idioma del elemento
6 xml:lang="codigo" Indica el idioma del elemento, aunque tiene más prioridad que el atributo anterior y es obligatorio si se incluye el atributo lang
7 dir Indica la dirección del texto
8 onclick Ejecuta la acción cuando se realiza un clic sobre el elemento
9 ondblclick Ejecuta la acción cuando se realiza un doble clic sobre el elemento
10 onmousedown Ejecuta la acción cuando se detecta el botón pulsado del ratón
11 onmouseup Ejecuta la acción cuando se detecta que se ha soltado el botón del ratón
12 onmousemove Ejecuta la acción cuando se detecta el movimiento del ratón sobre el elemento
13 onmouseout Ejecuta la acción cuando el ratón abandona el elemento
14 onmouseover Ejecuta la acción cuando se detecta que el ratón se sitúa sobre el elemento
15 onload Ejecuta la acción cuando se carga el documento
16 onunload Ejecuta la acción cuando se abandona el documento
17 onresize Ejecuta la acción cuando se ha modificado el tamaño de la ventana del navegador
18 onkeydown Ejecuta la acción cuando se detecta que la tecla esta pulsada
19 onkeyup Ejecuta la acción cuando se detecta que se ha soltado la tecla pulsada
20 onkeypress Ejecuta la acción cuando se pulsa una tecla
21 onblur Ejecuta la acción cuando el elemento pierde el foco bien sea a través del ratón o por navegación tabulada
22 onfocus Ejecuta la acción cuando el elemento obtiene el foco bien sea a través del ratón o por navegación tabulada
23 onchange Ejecuta la acción cuando el valor de un control ha sido modificado
24 onreset Ejecuta la acción cuando el formulario es restablecido a sus valores por defecto
25 onselect Ejecuta la acción cuando un usuario selecciona texto en un campo de texto
26 onsubmit Ejecuta la acción cuando el formulario es enviado
27 accesskey="letra" Establece una tecla de acceso rápido a un elemento HTML
28 tabindex="numero" Establece la posición del elemento en el orden de tabulación de la página (valor entre 0 y 32.767)
29 onfocus, onblur Controlan los eventos JavaScript que se ejecutan cuando el elemento obtiene o pierde el foco
